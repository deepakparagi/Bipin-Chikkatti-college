# Bipin Chikkatti College Website

A premium, elegant college website built with pure HTML5, CSS3, and Vanilla JavaScript featuring a "Subtle Minimal Classic Old-Money Premium" aesthetic.

## 🎓 About

This is the official website for **Bipin Chikkatti College**, part of the Chikkatti Group of Institutions founded by Prof. S. Y. Chikkatti in 1994. Located on a 7-acre lush green campus in Gadag, Karnataka, the college offers undergraduate and postgraduate programs in Science, Commerce, and Arts.

## ✨ Design Philosophy

The website embodies a timeless, sophisticated aesthetic inspired by:
- Ivy League university websites
- Luxury editorial design
- Heritage academic institutions
- High-end minimalist branding

### Color Palette
- **Cream/Off-white backgrounds** (#FAF8F3, #FFFFF0)
- **Deep charcoal/navy accents** (#2C2C2C, #1A2332)
- **Muted gold highlights** (#B8956A)

### Typography
- **Headings:** Playfair Display (Serif)
- **Body Text:** Inter (Sans-serif)
- Large, elegant, spacious typography with careful alignment

## 🚀 Getting Started

1. **Clone or download** the project
2. **Open `index.html`** in a web browser
3. **No build process required** - it's pure HTML/CSS/JS!

### Local Development

For a better development experience, use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000`

## 🎨 Customization

### Colors
Edit `css/variables.css` to change the color scheme:
```css
:root {
  --color-cream: #FAF8F3;
  --color-charcoal: #2C2C2C;
  --color-gold: #B8956A;
  /* ... more variables */
}
```

### Typography
Change fonts in `css/base.css`:
```css
@import url('https://fonts.googleapis.com/css2?family=Your+Font');
```

### Content
Update text content directly in the HTML files.
